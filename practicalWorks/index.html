Практическая работа 3.1

Задание 1

Напишите запрос на создание 6-7 новых автовладельцев и 5-6 автомобилей, каждому автовладельцу назначьте удостоверение и от 1 до 3 автомобилей.

  • Создание пяти новых автомобилей

  • Создание экземпляров владельцев и ассоциативных сущностей для связи владельцев с машинами

  • Создание экземпляров прав для владельцев

Задание 2

  • Вывести все машины марки “Toyota”

  • Найти всех водителей с именем “Павел”

  • Взяв любого случайного владельца и получить экземпляр его удостоверения в виде объекта модели Чтобы получить объект модели, необходимо использовать функцию get(), либо взять конкретный объект из QuerySet:

DriversLicense.objects.get(owner_id=drivers[0])

// or

queryset = DriversLicense.objects.filter(owner_id=drivers[0])
queryset[0]

  • Вывести всех владельцев черных машин

    CarOwner.objects.filter(cars__color="black").distinct()

  • Найти всех владельцев, чей год владения машиной начинается с 2021

Задание 3

  • Вывести дату выдачи самого старшего водительского удостоверения

  • Вывести самую позднюю дату владения машиной

  • Вывести количество машин для каждого водителя

  • Подсчитать количество машин каждой марки

    Car.objects.values("make").annotate(Count("id"))

  • Отсортировать всех автовладельцев по дате выдачи удостоверения

    CarOwner.objects.order_by("driverslicense__date_of_issue").distinct()
